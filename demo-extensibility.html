JavaScript API Examples

Add a Single Page

// Add a new page programmatically const newPage = GPTIdeasHub.addNewPage( "AI-Powered Content Creation", "Exploring how artificial intelligence is revolutionizing content creation workflows", "tech", ["AI", "Content Creation", "Automation"] );

Add Multiple Pages

// Add multiple pages at once const newPages = [ { title: "Sustainable Web Design", description: "Creating eco-friendly websites with minimal environmental impact", category: "tech", tags: ["Sustainability", "Web Design", "Green Tech"] }, { title: "Digital Nomad Lifestyle", description: "Tips and strategies for working remotely while traveling the world", category: "lifestyle", tags: ["Remote Work", "Travel", "Digital Nomad"] } ]; GPTIdeasHub.addMultiplePages(newPages);

Update Existing Page

// Update a page by ID GPTIdeasHub.updatePage(1, { title: "Updated Title", description: "Updated description", tags: ["Updated", "Tags"] });

Remove Page

// Remove a page by ID GPTIdeasHub.removePage(101);

Query Pages

// Get total page count const totalPages = GPTIdeasHub.getTotalPages(); // Get pages by category const techPages = GPTIdeasHub.getPagesByCategory('tech'); // Get pages by tag const aiPages = GPTIdeasHub.getPagesByTag('AI'); // Set items per page for pagination GPTIdeasHub.setItemsPerPage(50);

Features

  • Unlimited Pages: Add as many pages as you want - no 100 page limit
  • Dynamic Management: Add, remove, and update pages in real-time
  • Smart Pagination: Automatically handles large numbers of pages
  • Category Management: Organize pages by categories
  • Tag System: Flexible tagging for better organization
  • Search & Filter: Find pages quickly across all content
  • API Access: Full JavaScript API for programmatic control
  • Performance Optimized: Efficient rendering even with thousands of pages
