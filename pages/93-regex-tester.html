How to Use

  1. Enter Regex: Type your regular expression pattern
  2. Add Test Text: Enter the text you want to test against
  3. Set Flags: Choose appropriate regex flags (g, i, m, u)
  4. Test: Click the test button to see results
  5. Analyze: Review matches, groups, and performance

Regex Flags

  • g (Global): Find all matches, not just the first
  • i (Case Insensitive): Ignore case when matching
  • m (Multiline): Treat start/end anchors as line boundaries
  • u (Unicode): Enable full Unicode support

Common Patterns

  • Word characters: \w matches letters, digits, underscore
  • Digits: \d matches any digit (0-9)
  • Whitespace: \s matches spaces, tabs, newlines
  • Word boundaries: \b matches word boundaries
  • Quantifiers: + (1+), * (0+), ? (0 or 1)
